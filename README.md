# Web Application in Rust

## Details
- `diesel` for ORM
- `axum` for Web Application
- `dotenvy` for properties load
- `env_logger` for application logs
- `serde_json` for Json handling

## How to run

#### Build command
```shell
  cargo build
```
#### Run migrations
```shell
diesel migration run
```
#### Run command
```shell
  cargo run
```
#### Test command
```shell
  cargo test
```
